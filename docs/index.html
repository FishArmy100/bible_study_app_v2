Welcome the Ascribe Project Website!

Ascribe is a free and open source bible study app project which can be found here. It was built for the purpose of note taking, highlighting, and general Bible study that can be used offline. There are other apps and programs like Olive tree, Blue Letter Bible, and E-Sword. These are all amazing apps, however none of them fit the bill as being easy to use, offline, free/open source, and supports extensive note taking.

Currently Supported Features:

  • Entire KJV entirely offline
  • Searching words and sections
  • Custom highlights

Future Features (Lord willing):

  • Multiple open source versions:
    • ASV
    • WEB
    • BBE
    • YLT
  • Note taking system with cross references
  • Bible inserts
  • Bible printing
  • Markdown support:
    • Image and file imports
    • Chart building
    • Standard Github Markdown
  • And more!
