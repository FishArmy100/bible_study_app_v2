Searching

Search Methods

You can search using two primary methods: section searches and word searches. Section searches are in the format: Book chapter:start-end, for example: Genesis 1:3-5. Book names can be shortened, like Gen, Num, jn, etc. Also, capitalization does not matter. The verse range can be omitted (ie, 3-5) if you want to just get a whole chapter, or a singular verse can be listed. The : and the - can be omitted for quick searching.

Below is some example searches:
  • Genesis 5
  • Job 40:3
  • Song of Solomon 2:3-7
  • 1 Cor 2:9

Markdown Formatting

Markdown is a lightweight markup language for creating formatted text using a plain-text editor. It is commonly used for README files, documentation, and online writing. Ascribe uses it inside of notes and highlight descriptions to allow for easy to format text, without having to use a complicated editor, and allowing for cross platform note taking, if the user would want to copy the JSON save to another program.

Basic Syntax

  • Headings: Use # for headings. Example: # Heading 1
  • Bold: Use **bold** for bold text.
  • Italic: Use *italic* for italic text.
  • Lists: Use - or * for bullet points.
  • Links: [Link text](http://example.com)
  • Images: ![Alt text](image.jpg)
  • Code Blocks: Use triple backticks (```) for code blocks.

Example


# My Markdown Page

## Introduction

This is a **bold** text and this is an *italic* text.

- Item 1
- Item 2

[Click here](http://example.com) for more info.

You can find more information about markdown at this link.
